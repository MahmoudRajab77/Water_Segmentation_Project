# Water Segmentation using Multispectral and Optical Data

## Project Overview
U-Net implementation for water body segmentation from satellite imagery.

## Data
- **Images**: 12-channel satellite images (.tif), 512×512 pixels
- **Labels**: Binary water masks (.png), 128×128 pixels

## Setup
```bash
pip install -r requirements.txt
python visualize.py
python main.py
